This free gasoline cost calculator evaluates the fuel cost of a journey using several units of measurement depending on fuel efficiency, distance, and gas price.

[https://www.calculator.io/gas-cost-calculator/](https://www.calculator.io/gas-cost-calculator/)

== Usage ==

Add the Gas Cost Calculator shortcode to your page, post or sidebar:

`[ci_gas_cost_calculator]`

Add the following code to your template where you would like the Gas Cost Calculator to appear:

`<?php display_ci_gas_cost_calculator(); ?>`

== Installation ==

1. Upload the Gas Cost Calculator /ci_gas_cost_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Gas Cost Calculator plugin through the "Plugins" menu in WordPress.
